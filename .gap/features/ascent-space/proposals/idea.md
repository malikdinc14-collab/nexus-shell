# Idea: Ascent Learning Space

## Vision
A specialized environment designed for self-improvement and technical mastery. It transforms Nexus Shell into an adaptive learning tutor.

## Core Components
1.  **Learner Level (HUD Expansion)**: The Status HUD will display the user's current "Level" or "Exp" based on completed tasks and code complexity.
2.  **Curriculum Pane**: A dedicated pane (Tmux Slot 11) that displays the current "Learning Curriculum" (Markdown files in `.ascent/`).
3.  **The Mastermind Composition**: A layout optimized for study:
    - **Top Left**: Curriculum/Guide.
    - **Top Right**: Multi-pane (Editor + Terminal).
    - **Bottom**: Progress HUD + Notes.

## Gamification
- **Mission Progress**: Real-time feedback in the HUD when a unit test passes or a feature is merged.
- **Level-Up Events**: Sound effects or TUI animations when crossing knowledge thresholds.
