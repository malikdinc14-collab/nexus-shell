# REQ-002: Dynamic Visualisation & Monitoring

## Overview
Enable real-time visualisation of background processes, agent telemetry, and system performance through dynamic pane management.

## Functional Requirements
- **FR-01**: Support creation of temporary "Pop-up" panes for process monitoring.
- **FR-02**: Provide a dedicated "Intelligence Monitor" surface.
- **FR-03**: Allow switching between different TUI dashboards (htop, btm, mactop) within a managed pane.
- **FR-04**: Visualise LLM inference metrics (tokens/sec, latency) if available.
