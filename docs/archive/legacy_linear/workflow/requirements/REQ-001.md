# REQ-001: Local Intelligence & Visual Rendering

## Overview
Transform the local development environment into an "Intelligence OS" utilizing local models and rich visual feedback within the terminal.

## Functional Requirements
- **FR-01**: Support multi-model orchestration on Apple Silicon (M4 Pro).
- **FR-02**: Implement a "Scout" (lightweight) and "Expert" (heavyweight) dual-model architecture.
- **FR-03**: Integrate Letta (formerly MemGPT) for persistent, stateful memory.
- **FR-04**: Implement Recursive Language Model (RLM) logic with 1-layer recursion for long-context reasoning.
- **FR-05**: Provide a real-time hardware and model telemetry dashboard using `mactop`.
- **FR-06**: Support rich media rendering in the terminal (Mermaid diagrams, images) using iTerm2 protocols.
- **FR-07**: Automate the distillation of session history into long-term memory (MemEvolve).

## Technical Constraints
- **CON-01**: Must run locally on macOS (Darwin/ARM64).
- **CON-02**: Must stay within the 48GB Unified Memory limit (target ~40GB max usage).
- **CON-03**: Use iTerm2 as the primary terminal emulator for visual rendering.
- **CON-04**: Adhere to the Nexus-Shell/Parallax project structure.
