### 🕹️ THE DUAL-LAYER NAVIGATION
Nexus uses a hierarchical tab system to keep your focus sharp:

1. **WORKSPACE TABS (Nexus-Tabs)**
   - Displayed at the **TOP STATUS BAR**.
   - These represent full window layouts.
   - `Alt-Shift-[` / `Alt-Shift-]` to cycle layouts.

2. **TOOL TABS (Pane-Tabs)**
   - Displayed in the **PANE BORDERS**.
   - These represent specific tools (Chat, Shell, Logs) within a pane.
   - `Alt-[` / `Alt-]` to cycle tools.
